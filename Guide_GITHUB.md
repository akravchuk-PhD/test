# GitHub Guide

### For command line navigation:

To obtain the repository on your machine:

```
> git clone https://github.com/<USR>/<REPO>.git

Pssd: <PSSD>
```

To check status (what needs to be done to the repository):

```
> git status
```

To push:

```
> git add test.R

> git commit -m "Updates to project."

> git push

```

To pull:

